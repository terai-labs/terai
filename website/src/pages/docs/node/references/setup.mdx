# `setupTerai`

The `setupTerai` function initializes the Terai runtime with your configuration. It sets up the loader, default locale, and global formatting options. This must be called before using any other Terai functions.

```ts
import { setupTerai } from '@terai/node'

setupTerai({
  defaultLocale: 'en',
  loader: (locale) => import(`./locales/${locale}.json`),
})
```

## Parameters

The `setupTerai` function accepts a single configuration object with the following properties:

### `loader`

- **Type**: `(locale: string, chunkId: string) => Promise<Dictionary>`
- **Required**: Yes

A function responsible for loading the translation dictionary for a given locale.

### `defaultLocale`

- **Type**: `string`
- **Required**: No
- **Default**: `'en'`

The initial locale to use. This sets the global locale state.

### `format`

- **Type**: `GlobalFormat`
- **Required**: No
- **Default**: `{}`

Global configuration for formatting options.

## Return Value

`void`. This function initializes the internal global state and does not return anything.
