# `getFormat`

The `getFormat` function returns formatting utilities for the current global locale.

```ts
const format = getFormat()
```

## Parameters

None. It uses the current global locale set by `setupTerai` or `setLocale`.

## Return Value

Returns a `Format` object.

## Return Value

Returns a `Format` object containing the following methods:

### `format.number`

Formats numbers for currency, percentages, decimals, etc.

- **Signature**: `(value: number, options?: Intl.NumberFormatOptions) => string`

```ts
format.number(1234.56, { style: 'currency', currency: 'USD' })
// Output: "$1,234.56"
```

### `format.date`

Formats dates and times.

- **Signature**: `(value: Date | number, options?: Intl.DateTimeFormatOptions) => string`

```ts
format.date(new Date(), { dateStyle: 'full' })
// Output: "Tuesday, November 19, 2025"
```

### `format.list`

Formats a list of items (conjunctions or disjunctions).

- **Signature**: `(value: Iterable<string>, options?: Intl.ListFormatOptions) => string`

```ts
format.list(['Apple', 'Banana', 'Orange'], { type: 'conjunction' })
// Output: "Apple, Banana, and Orange"
```

### `format.displayName`

Returns the localized display name for a region, language, script, or currency.

- **Signature**: `(value: string, options?: Intl.DisplayNamesOptions) => string | undefined`

```ts
format.displayName('US', { type: 'region' })
// Output: "United States"
```

### `format.relativeTime`

Formats a relative time value (e.g., "5 minutes ago", "in 2 days").

- **Signature**: `(value: number, unit: Intl.RelativeTimeFormatUnit, options?: Intl.RelativeTimeFormatOptions) => string`

```ts
format.relativeTime(-1, 'day')
// Output: "yesterday"

format.relativeTime(5, 'minute', { style: 'short' })
// Output: "in 5 min."
```
