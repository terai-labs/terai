# `getDictionary`

Retrieves the dictionary for a specific locale from the store.

```ts
import { getDictionary } from '@terai/node'

const dict = getDictionary('en')
```

## Parameters

### `locale`

- **Type**: `string`
- **Required**: Yes

The locale to retrieve the dictionary for.

## Return Value

Returns the `Dictionary` object if loaded, or `undefined` if not found.

---

# `getDictionaries`

Retrieves all loaded dictionaries from the store.

```ts
import { getDictionaries } from '@terai/node'

const allDicts = getDictionaries()
```

## Return Value

Returns a `Dictionaries` object containing all loaded dictionaries indexed by locale.
