# `getTs`

The `getTs` function retrieves a translation function (`ts`). It uses the global locale by default but can be overridden with a specific locale.

```ts
// Uses global locale
const ts = await getTs()

// Uses specific locale
const tsEs = await getTs({ locale: 'es' })
```

## Parameters

### `options`

- **Type**: `{ locale?: string, chunkId?: string }`
- **Required**: No

Configuration options:
- `locale`: Override the global locale.
- `chunkId`: Specify a chunk to load (defaults to the locale name).

## Return Value

Returns a `Promise` that resolves to a `ts` tag function.
