# `getTs` Reference

Function to retrieve the `ts` translation function for a specific locale. It is returned by `setup`.

```ts
const ts = await getTs({ locale: 'en' })
console.log(ts`Hello, world!`)
```

## Parameters

### options

- **Type**: `{ locale: string }`
- **Required**

Object containing the locale for which to get the translation function.

## Return Value

Returns a Promise that resolves to the `ts` function.

### ts

- **Type**: `` (strings: TemplateStringsArray, ...values: any[]) => string ``

The translation function that uses tagged template literals to translate messages. It automatically:
- Extracts the message and normalizes it
- Looks up the translation in the loaded dictionary
- Interpolates any variables passed in the template

**Example**:

```ts
const ts = await getTs({ locale: 'es' })

// Simple translation
const greeting = ts`Hello, world!`

// With variables
const name = 'Juan'
const personalGreeting = ts`Hello, ${name}!`
```
